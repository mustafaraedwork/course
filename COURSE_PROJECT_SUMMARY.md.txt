## 🛠️ التقنيات المستخدمة

### Frontend
- **Next.js 15.4.5** - إطار عمل React
- **Tailwind CSS v4** - للتصميم السريع والمرن
- **TypeScript** - للأمان والوضوح

### Backend & Database
- **Supabase** - قاعدة البيانات والمصادقة
- **PostgreSQL** - قاعدة البيانات الأساسية
- **Row Level Security** - لحماية البيانات

### التصميم
- **Meta/Facebook Theme** - نظام ألوان مميز
- **Primary Color**: #1877f2 (Facebook Blue)
- **Secondary Color**: #42b883 (Success Green)
- **Responsive Design** - يعمل على جميع الأجهزة
